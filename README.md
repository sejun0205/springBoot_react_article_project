# board-react-project

게시판 실습 프로젝트 인덱스 저장소입니다.  
백엔드/프론트엔드는 아래 개별 저장소에서 관리합니다.

## Repository Links

- 백엔드 (Spring Boot, MyBatis, MySQL)  
   [react_article_backend](https://github.com/sejun0205/react_article_backend)

- 프론트엔드 (React, TypeScript, React Router)  
   [react_article_frontend](https://github.com/sejun0205/react_article_frontend)

## Features

- 회원가입 / 로그인 / 로그아웃 (세션 기반)
- 게시글 CRUD
- 댓글 CRUD
- 작성자 본인만 수정/삭제 가능

## Tech Stack

- Backend: Java, Spring Boot, MyBatis, MySQL
- Frontend: React, TypeScript, Vite, React Router
